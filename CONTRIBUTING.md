# Contributing to Aqualyx AI

Thank you for your interest in contributing to **Aqualyx AI**!  
We welcome contributions of any kind: **code, documentation, bug reports, feature suggestions, or design improvements**.

---

## How to Contribute

1. **Fork the repository**  
2. **Clone your fork**
- git clone https://github.com/ankitkumar04100/Aqualyx-AI.git
3. **Create a new branch**
- git checkout -b feature/your-feature
4. **Make your changes**
- (code, docs, dataset, or frontend)
5. **Test your changes**
- Backend + ML: run scripts/train_model.py and backend/main.py
- Frontend: run npm install and npm start
6. **Commit and push**
- git add .
- git commit -m "Add feature X or fix issue Y"
- git push origin feature/your-feature
7. **Open a Pull Request (PR)** to the main branch

---

## Guidelines

- Follow PEP8 for Python and standard best practices for JavaScript/React
- Document all functions and components clearly
- Include screenshots or sample outputs if relevant
- Use descriptive commit messages
- Be respectful, constructive, and collaborative in discussions

---

## Reporting Issues

- Open issues for bugs, suggestions, or feature requests
- Provide clear steps to reproduce the problem
- Include screenshots, error messages, or dataset examples when possible

---

## Code Review & Approval

- All pull requests will be reviewed by the maintainer
- PRs must pass basic testing before merging
- Ensure code readability, documentation, and correctness

## Acknowledgments

Thank you for helping improve Aqualyx AI!
Your contributions make it more robust, professional, and hackathon-ready.
